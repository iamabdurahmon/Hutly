# Hutly

A modern, professional business website featuring a clean design with dark mode support and responsive navigation.

## Features

- **Responsive Design** - Fully optimized for desktop, tablet, and mobile devices
- **Dark Mode** - Toggle between light and dark themes for enhanced user experience
- **Modern UI** - Professional interface with smooth animations and transitions
- **Mobile-Friendly Navigation** - Hamburger menu with smooth interactions
- **Section Navigation** - Quick links to Home, Projects, About Us, and Services

## Project Structure

```
Hutly/
├── index.html      # Main HTML file
├── style.css       # Styling and theme management
├── script.js       # JavaScript functionality
├── image/          # Images and assets (logo, favicon, etc.)
└── README.md       # This file
```

## Technologies Used

- **HTML5** - Semantic markup
- **CSS3** - Responsive styling with CSS variables for theme management
- **JavaScript** - Interactive features and DOM manipulation
- **Remix Icon** - Icon library for UI elements

## Quick Start

1. Clone or download the repository
2. Open `index.html` in your web browser
3. No build process or installation required

## Usage

### Dark Mode Toggle

Click the moon/sun icon in the header to switch between light and dark modes.

### Mobile Navigation

Use the hamburger menu on mobile devices to access the navigation menu.

### Navigation Links

- **Home** - Landing section
- **Projects** - Portfolio showcase
- **About Us** - Company information
- **Services** - Service offerings
- **Contact Us** - Get in touch

## Browser Support

Works on all modern browsers including Chrome, Firefox, Safari, and Edge.
